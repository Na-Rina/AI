리눅스 명령어 교정 프로젝트
## 개요

리눅스 명령어를 입력할 때 발생할 수 있는 오타를 자동으로 수정해주는 프로그램입니다. 위 프로그램을 통해 권한 부여와 같은 중요한 작업에서 인간의 실수로 발생할 수 있는 오타를 교정해주어 클라이언트가 원하는 결과를 도출할 수 있습니다. 

## 기능

- 오타 수정을 통해 사용자가 원하는 결과를 출력
- 단축키를 통해 프로세스 종료, 권한부여 기능에 대한 편의성 제공
